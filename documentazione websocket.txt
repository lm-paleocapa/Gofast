Json ricevuti dal websocket server

id <1> Serve per far associare al programma l'utente ad un socket.
-campi da compilare     {id, user0}
-significato            {1, utente che si è connesso}
-variabile 		{int, string}

id <2> reindirizzamento del messaggio
-campi da compilare {id,text, user0, user1, date, image}
-significato        {2, testo del messaggio, utente di partenza, utente destinatario, data di invio (), immagine in base64}
-variabile          {int, string, string, string, DateTime, string}

id <3> logout Profilo
-campi da compilare {id, date}
-significato 	    {3, data di disconessione (yyyy/MM/dd HH-mm) => in questo formato}
-variabile  	    {int, DateTime}



Json ricevuti dal web socket client

id <1> Serve al client che riceve il messaggio per sapere chi gli ha inviato il messaggio e quando è stato inviato
-Campi da compilare {id = 1, text, user0, date}
-significato        {1, testo, utente che ha inviato inviato il messaggio, data di partenza del messaggio}
-variabile          {int, string, string, DateTime}

id <2> Serve al client per definire gli utenti online e l'ultimo accesso 
-campi da compilare {state}
-variabile   	    {string}
